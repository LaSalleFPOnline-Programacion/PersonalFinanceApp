# PersonalFinanceApp - Midterm

## Descripción
Aplicación Java para la **gestión personal de finanzas**. Permite:
- Registrar gastos con categoría y método de pago.
- Registrar ingresos con origen y fecha.
- Consultar resumen financiero: saldo total y gastos por categoría.
- Mantener un historial organizado de todos los movimientos.
