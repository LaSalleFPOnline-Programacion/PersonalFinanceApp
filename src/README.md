# PersonalFinanceApp - Midterm

## Descripción
Aplicación Java para la gestión personal de finanzas. Permite:
- Registrar gastos con categoría y método de pago.
- Registrar ingresos con origen y fecha.
- Consultar resumen financiero: saldo total y gastos por categoría.

## Funcionalidades
1. Menú principal con submenús de gastos, ingresos y resumen.
2. Alta, modificación y eliminación de gastos e ingresos.
3. Validaciones de importes, fechas y campos obligatorios.
4. Gestión de errores en entradas por consola.

## Tecnologías
- Java 17
- IntelliJ IDEA
- Git y GitHub para control de versiones
